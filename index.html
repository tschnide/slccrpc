SLCC RPC

Salt Lake Community College Raspberry Pi Club

RPC exists to learn how to build projects using a Raspberry Pi computer. We want to build projects that help us learn to work as a team, communicate well, solve unique problems, and take initiative. We recognize that these are qualities that are important in the the programming industry, but hard to teach in a classroom and we will create an environment where we can learn them before we have graduated so we will be better prepared when we move forward from Salt Lake Community College.
